# 高级前端 UI（去「AI 味」）

做网页 / 组件 / 落地页时，产出**有设计感、像真人精修过**的界面，而不是一眼假的通用 AI 模板（紫蓝渐变 + 居中卡片 + emoji 堆砌那一套）。

## 避开的「AI 味」
- 千篇一律的紫→蓝线性渐变、霓虹发光、毛玻璃滥用。
- 所有东西居中、卡片圆角阴影一个样、间距均匀到呆板。
- 标题配 emoji、文案空话（"赋能 / 一站式 / 革命性"）。
- 字体只用系统默认、配色只用纯黑纯白纯灰。

## 怎么做出高级感
1. **排版优先**：一套清晰的字号层级（display / h1-h3 / body / caption），克制的字重对比，行高与字距调过。选一款有性格的字体搭配。
2. **配色有主张**：定一个主色 + 一个强调色 + 中性色阶，别堆七彩。深色界面用炭黑而非纯黑，浅色用暖白而非纯白。
3. **留白与节奏**：用不均匀的间距制造层次（区块之间大留白，组内紧凑）。8pt 网格起步。
4. **细节收口**：hover / focus / active 状态、过渡动画克制（150-250ms）、边框用低对比度、阴影柔且有方向。
5. **真实内容**：用贴合主题的真实文案和数据占位，别用 "Lorem ipsum" 和 "标题标题"。

## 约定
- 响应式：移动优先，断点处重排不是简单缩放。
- 可访问性：语义标签、对比度达标、键盘可达。
- 自包含输出优先（单文件 HTML 内联样式），便于预览与交付。
